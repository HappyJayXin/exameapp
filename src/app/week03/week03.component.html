另外建立副程式

public  int cal(int no){
    int sum =0;
    for(int i=1;i<=no;i++)
    {
        sum +=i;
    }
    return  sum;
}

SharedPreferences 完整用法與檔案檢視

  

    宣告
      SharedPreferences sp;
     建立物件
      sp=getSharedPreferences("myData2", MODE_PRIVATE);
     
     新增資料
                     sp.edit().putString("name","蘇建郡abc")
                             .putInt("age",90)
                             .putString("address","台南市")
                             .commit();
     另一種寫法
     //                SharedPreferences.Editor editor = sp.edit();
     //                editor.putString("a","123");
     //                editor.putString("b","456");
     //                editor.putInt("c",11);
     //                editor.commit();
     移除特定資料
           sp.edit().remove("name").commit();

     //                        .remove("age")
     //                        .remove("address")
     //                        .commit();
     整個資料清除
         sp.edit().clear().commit();
